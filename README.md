## Development

#### Getting Started

1. Clone the repository.

    ```shell
    git clone https://github.com/Austin-Imbastari/r34-scene-1.git
    cd r34-scene-1
    ```


1. Install all dependencies with NPM.

    ```shell
    npm install
    ```
    
1. Run the webpack development server.

    ```shell
    npm start
    ```
